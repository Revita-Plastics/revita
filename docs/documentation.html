ReVita — Eenvoudige en visuele documentatie

Deze pagina legt op een simpele manier uit wat elk belangrijk deel van het project doet en hoe je veelvoorkomende taken uitvoert (foto toevoegen, naam wijzigen, lokaal testen, publiceren). Geen technische voorkennis vereist.

Wat is dit project?

Het is een kleine website gemaakt met React en Vite. React is een manier om webpagina's op te bouwen met onderdelen (zoals LEGO-blokjes). Vite is het gereedschap dat alles snel bij elkaar zet.

Belangrijkste mappen:

  • /src — hier staat de code (knoppen, pagina's, producten, afbeeldingen).
  • /src/images — je eigen foto's (bijv. plantenpot.webp).
  • /build — output na 'bouwen' (klaar om online te zetten).
  • /docs — deze documentatie.

Veelvoorkomende taken — stap voor stap

1) Eigen foto toevoegen (zodat die zichtbaar is op de website)

Stap 1: kopieer je afbeelding naar /src/images (bijv. mijn-pot.webp).

Stap 2: open src/App.tsx en voeg bovenaan een regel toe om de afbeelding te importeren: 

import MijnPot from "./images/mijn-pot.webp";

Stap 3: in de productenlijst zet je bij het product image: MijnPot

Waarom dit? Vite zorgt dan zelf dat de afbeelding op de juiste plek in de uiteindelijke website terechtkomt.

2) De naam van de site veranderen (bijv. van "ReVita Upcycled" naar "ReVita")

Open en wijzig deze bestanden:

  • index.html — wijzig de <title> naar <title>ReVita</title>
  • src/components/Navigation.tsx — wijzig de tekst die het merk toont naar "ReVita"
  • src/components/AboutPage.tsx — pas kop en tekst aan indien nodig

Tip: zoek in het project naar "ReVita Upcycled" en vervang dat door "ReVita". 

Select-String -Path .\* -Pattern "ReVita Upcycled" -SimpleMatch -List

3) Site lokaal testen

Als je Node.js hebt geïnstalleerd, kun je in de projectmap dit commando gebruiken om een lokale preview te starten: 

npm run dev

Open dan je browser op http://localhost:5173 (of het adres dat in de terminal verschijnt).

4) Site online zetten (GitHub Pages)

Algemene stappen:

  1. Maak een GitHub-repository (of gebruik de bestaande: SepEmmers/revita).
  2. Pushe je code naar GitHub (git push).
  3. Gebruik het bouw- en publicatiescript in package.json: npm run deploy. Dit bouwt de site en publiceert de inhoud van de build-map naar GitHub Pages.

Als je wilt, kan ik een automatische GitHub Action toevoegen zodat elke keer wanneer je naar main pusht de site automatisch wordt gebouwd en gepubliceerd.

Mijn afbeelding verschijnt nog steeds niet — wat nu?

Controleer of je afbeelding in /src/images staat en dat je hem importeert in App.tsx. Als je een externe URL gebruikt, controleer of de link werkt.
